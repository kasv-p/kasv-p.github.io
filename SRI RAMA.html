About Project

We are doing this project under software systems lab course (CS 213) which is being taught by Professor Tamal Das

  • Abstract - We are living in the age of Social Networking like Facebook, LinkedIn, Google ++ etc. The objective is to develop a social networking application
  • Title - Social Networking
  • Website Name - catchup

Basic features of the Website

  • User can register into the application with their name, email id and password
  • Registered user may be able to login into the application
  • There should be options to get the basic information like date of birth, address, phone no,education, upload his/her picture, professional information, hobby etc
  • After login, user should be able to see their profile information
  • One should be able to add new friends
  • One should be able to see the basic profile information of their friend
  • One should be able to send messages to their friends
